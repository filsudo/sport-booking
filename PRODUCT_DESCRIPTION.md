# Product Description Copy

## Short Description
SportBook is a production-ready sports booking template for Next.js and Supabase.  
It includes end-to-end booking flow, resource-based scheduling, protected admin panel, and white-label setup for fast launch.

## Key Features
- Full booking flow: service, date, resource, time, customer details, success page
- Resource-based booking for courts, tables, trainers, and group sessions
- Availability management with booked/unavailable/past slot states
- Double-booking protection on API + database level
- Admin panel for bookings, services, resources, and availability
- EN-first UI with SK demo mode and language switcher
- Mobile-friendly UI for booking and admin workflows

## Who This Is For
- Sports centers
- Tennis and badminton clubs
- Table tennis venues
- Personal training studios
- Agencies building white-label booking websites

## What Is Included
- Source code (Next.js + Supabase)
- SQL schema, RLS policies, seed data
- Setup guide, customization guide, deployment checklist
- Placeholder screenshots and package docs

## What Is Not Included
- Payment gateway integration
- SMS/WhatsApp notifications
- Managed hosting account
- Custom feature development

## Technical Requirements
- Node.js 20+
- Supabase account
- Vercel account (or any Next.js-compatible hosting)

## Support Scope
- Setup and template usage support
- Bug-fix support for included features
- No custom implementation work included by default
