# Support

## What Is Included
- bug fixes for template setup issues
- guidance for SQL/import problems
- clarification of docs and configuration

## What Is Not Included
- custom feature development
- third-party paid service setup (Stripe, Twilio, etc.)
- full custom design work

## Contact
- Email: `support@your-domain.com`
- Response target: 1-2 business days

## What To Send In A Support Request
- error message screenshot
- affected page/route
- steps to reproduce
- relevant environment values (without secrets)
